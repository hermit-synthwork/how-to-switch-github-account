# Handy Commands

A small place to keep useful commands that are easy to read and copy.

## Git Identity

Set your Git username for this repo. Replace the example with your actual name:

```bash
git config user.name "your name"
```

Set your Git email for this repo so commits use the right address. Replace the example with your actual email:

```bash
git config user.email "your@email.com"
```

## GitHub CLI

Switch the GitHub CLI to another account that is already signed in:

```bash
gh auth switch --user "user.name"
```

Check the authentication status for configured GitHub CLI hosts and accounts:

```bash
gh auth status
```
